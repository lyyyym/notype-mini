# NoType Mini 测试用例

本文档包含 NoType Mini 的端到端手动测试用例，用于验证核心功能和常见回归点。

## 环境准备

1. `git pull origin main` 拉取最新代码。
2. `npm install`
3. `npm run tauri dev`
4. 首次运行授予**麦克风**和**辅助功能**权限。
5. 在设置中填入有效的 DashScope API Key 和 LLM API Key（测试连接按钮应显示成功）。

---

## 1. 配置保存（快捷键未变化）

**步骤：**

1. 打开主窗口设置页。
2. 不修改任何快捷键，只修改 LLM Model 或剪贴板阈值。
3. 点击「保存配置」。

**期望：** 底部状态显示「配置已保存」，无快捷键相关错误弹窗/崩溃。

**回归点：** 修复了重复注册 macOS 全局热键导致的 `RegisterEventHotKey failed for Period` 错误。

---

## 2. 配置保存（快捷键变化）

**步骤：**

1. 把录音快捷键改为 `Command+Comma`。
2. 点击「保存配置」。
3. 在新快捷键上录音一次。
4. 再改回 `Command+Period`。

**期望：** 每次保存都成功，旧快捷键不再触发，新快捷键生效。

---

## 3. 普通录音转写

**步骤：**

1. 打开备忘录/微信输入框，放置光标。
2. 按住 `⌘+.` 说：「明天下午三点开会」。
3. 松开快捷键。

**期望：** 1~3 秒后输入框出现整理后的文字，如「明天下午三点开会。」主窗口历史记录增加一条类型为「录音」的条目。

---

## 4. 连续录音模式

**步骤：**

1. 设置中把录音模式切换为「连续录音」。
2. 按一次 `⌘+.` 开始录音，说一段话。
3. 再按一次 `⌘+.` 结束。

**期望：** 文字被输入，历史记录增加条目。

---

## 5. 取消录音

**步骤：**

1. 按住 `⌘+.` 开始录音。
2. 按 `Esc`。

**期望：** 录音气泡消失，没有文字输入，历史记录不增加。

---

## 6. 语音编辑（有选中文本）

**步骤：**

1. 在备忘录中输入：「这个东西很好」。
2. 选中「这个东西很好」。
3. 按住 `⌘+Option+.` 说：「改得正式一点」。
4. 松开。

**期望：** 选中文本被替换为更正式的表达，如「该产品表现出色。」历史记录增加类型为「编辑」的条目。

**回归点：** 修复了 `edit_pipeline` 中 `.expect("创建 Enigo 失败")` 导致的应用退出。

---

## 7. 语音编辑（无选中文本）

**步骤：**

1. 在输入框放置光标，不选中任何文字。
2. 按住 `⌘+Option+.` 说：「写一句问候语」。
3. 松开。

**期望：** 在光标处插入一句问候语，历史记录增加「编辑」条目。

---

## 8. 个人词典替换

**步骤：**

1. 设置页个人词典添加：`from=拉斯特`, `to=Rust`。
2. 保存配置。
3. 录音说：「拉斯特是一门好语言」。

**期望：** 输出中出现「Rust是一门好语言」，而不是「拉斯特」。

---

## 9. 个人词典最长匹配

**步骤：**

1. 设置页同时添加两个条目：
   - `from=拉斯特`, `to=Rust`
   - `from=拉斯特级`, `to=RustLevel`
2. 保存配置。
3. 录音说：「拉斯特级战舰」。

**期望：** 输出「RustLevel战舰」，优先匹配更长的 `拉斯特级`。

---

## 10. 剪贴板回退

**步骤：**

1. 设置中把「剪贴板回退字数阈值」设为 10。
2. 录音说一段超过 10 个字的文字。

**期望：** 文字通过剪贴板粘贴方式输入（可观察到输入前有轻微延迟，且不会有逐字输入效果）。

---

## 11. 系统托盘常驻

**步骤：**

1. 启动应用。
2. 点击主窗口左上角红色关闭按钮。
3. 观察 Dock/菜单栏托盘图标。
4. 右键托盘图标 → 「显示主窗口」。

**期望：** 应用未退出，托盘图标仍在；点击「显示主窗口」后主窗口重新出现。

---

## 12. 历史记录与统计拆分

**步骤：**

1. 执行一次普通录音。
2. 执行一次语音编辑。
3. 查看主窗口「统计」区域。

**期望：** 录音累计次数/字数和编辑累计次数/字数分别显示，且均增加 1。

---

## 13. 配置持久化

**步骤：**

1. 修改设置（例如换模型、添加词典条目、改阈值）。
2. 保存配置。
3. 右键托盘退出应用。
4. 重新 `npm run tauri dev`。

**期望：** 重新打开后所有设置与上次一致。

---

## 14. 无效配置校验

**步骤：**

1. 把录音快捷键和编辑快捷键都设为 `Command+Period`，点击保存。
2. 把录音快捷键设为 `Escape`，点击保存。
3. 在个人词典中添加两条 `from=拉斯特` 的条目，点击保存。

**期望：** 三次保存都应失败并给出明确错误提示。

---

## 15. 录音与编辑互斥

**步骤：**

1. 按住 `⌘+.` 开始录音。
2. 在录音过程中尝试保存配置或再次按编辑快捷键。

**期望：** 保存配置提示「请先结束当前录音或编辑再修改设置」；编辑快捷键在录音中不触发新会话。

---

## 运行 Rust 单元测试

```bash
cd src-tauri
cargo test
```

所有测试通过是发布前必需条件。
